Engineered for Security and Reliability

Features that make DeviceGuardian AI suitable for enterprise telemetry monitoring

📁

Offline Resiliency

If network connection drops, telemetry logs are immediately cached to a local SQLite queue and synced automatically once connection returns.

⚙️

Silent Execution

Runs as a lightweight service in the background. The system tray allows pausing, manual syncs, and status reports in one-click.

🛡️

Secure JWT Auth

Supports robust token-based credentials authentication with custom encryption headers on every upload endpoint.

⚙️ Installation & Run Instructions

1

Download & Move

Download the DeviceGuardianAgent.exe binary and place it in a dedicated directory on your laptop.

2

Run as Administrator

Right-click the executable and select "Run as administrator" to enable motherboard thermal sensor access.

3

Link Your Account

On startup, a secure dialog will pop up. Enter your portal email (e.g. sagar@gmail.com) to instantly link this device.

4

Monitor Real-Time

The agent will run silently in your System Tray. Check your dashboard dropdown to select the device and view live analytics!
